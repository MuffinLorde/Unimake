# UNIMAKE (Currently completely being reworked)

UNIMAKE (Universal Maker), formally known as (Compile Prompt Initialiser), is a tool developed in 2024 by MuffinLorde (Liwa-Lam Ncanywa) to run files using a short command rather than multiple long commands used for all programming languages. It's also super useful for making PasteBin like Devnotes.
